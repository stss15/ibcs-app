IB Computer Science

Theme A: Concepts of Computer Science

Welcome. This unit explores the fundamental concepts underpinning modern computing systems. We'll start with the smallest parts—the logic gates—and build up to large, complex networks.

1.1 System Fundamentals

A computer system is defined as a combination of hardware and software that processes data. Understanding the components and their interactions is key to advanced problem-solving. This includes the CPU, primary memory, secondary storage, and I/O devices.

1.2 Binary Representation

All data in a computer is stored and processed using the binary system (base 2), which uses only two digits: 0 and 1. This is because the fundamental hardware components, transistors, can only exist in one of two states (on or off).

Key Definition

Bit: The smallest unit of digital information, representing either a 0 or a 1. Eight bits make up one byte.
