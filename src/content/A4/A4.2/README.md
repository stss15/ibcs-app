# A4.2 路 Data preprocessing (HL only)

### A4.2.1 路 Describe data cleaning significance
- Handle outliers, duplicates, incorrect/irrelevant data, missing values (imputation, deletion, predictive models).
- Normalisation and standardisation.

### A4.2.2 路 Describe feature selection
- Strategies: filter, wrapper, embedded methods; retain informative attributes.

### A4.2.3 路 Describe dimensionality reduction importance
- Address the curse of dimensionality: overfitting, complexity, sparsity, distance metrics, visualisation, memory usage.
- Preserve essential information while reducing variables.
