# A4.3 · Machine learning approaches (HL only)

### A4.3.1 · Linear regression for continuous outcomes
- Relationship between predictor and response; slope/intercept; r² for fit quality.

### A4.3.2 · Classification techniques
- K-Nearest Neighbours and decision trees for discrete categories; examples in recommender systems and diagnosis.

### A4.3.3 · Hyperparameter tuning and evaluation
- Metrics: accuracy, precision, recall, F1.
- Concepts: overfitting, underfitting, impact of tuning.

### A4.3.4 · Clustering techniques
- Group data based on feature similarity; apply to customer segmentation and other contexts.

### A4.3.5 · Association rule learning
- Discover relationships between attributes; apply to domains such as crime analysis or retail.

### A4.3.6 · Reinforcement learning foundations
- Agent, environment, actions, states, rewards, policies; exploration vs exploitation; cumulative reward.

### A4.3.7 · Genetic algorithms
- Pipeline: population, fitness, selection, crossover, mutation, evaluation, termination.
- Example: route optimisation (travelling salesperson).

### A4.3.8 · Artificial neural networks (ANNs)
- Explain perceptrons, weights, bias, activation functions, outputs.
- Multi-layer perceptrons (MLPs) with input, hidden, output layers.

### A4.3.9 · Convolutional neural networks (CNNs)
- Architecture: convolution, activation, pooling, fully connected, outputs.
- Discuss design choices (layers, kernel size, stride, activation, loss) and their impact.

### A4.3.10 · Model selection and comparison
- Match algorithms to problem characteristics; evaluate variability in performance.
