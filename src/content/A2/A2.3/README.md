# A2.3 · Data transmission

### A2.3.1 · Describe different types of IP addressing
- IPv4 vs IPv6; public vs private; static vs dynamic; network address translation (NAT).

### A2.3.2 · Compare media for data transmission
- Wired: fibre optic, twisted pair. Wireless transmission.
- Evaluate bandwidth, installation complexity, cost, range, interference, attenuation, reliability, security.

### A2.3.3 · Explain packet switching
- Segmenting data into packets; headers and control info; reassembly at destination.
- Roles of switches and routers.

### A2.3.4 · Explain static vs dynamic routing (HL only)
- Processes, advantages, disadvantages.
- Consider configuration, maintenance, complexity, resources, convergence, scalability, network size.
