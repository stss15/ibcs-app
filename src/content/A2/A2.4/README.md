# A2.4 · Network security

### A2.4.1 · Discuss the effectiveness of firewalls
- Filtering via whitelists, blacklists, rules; role of NAT; strengths and limitations.

### A2.4.2 · Describe common network vulnerabilities (HL only)
- Examples: DDoS, insecure protocols, malware, MitM, phishing, SQL injection, XSS, unpatched software, weak authentication, zero-day exploits.

### A2.4.3 · Describe common network countermeasures (HL only)
- Measures: CSP, password policies, DDoS mitigation, email filtering, encrypted protocols, input validation, IDS/IPS, MFA, SSL/TLS, software updates, VPNs, MAC filtering.

### A2.4.4 · Describe encryption and digital certificates
- Symmetric vs asymmetric cryptography.
- Role of digital certificates, public/private keys, key management.
