# A2.1 · Network fundamentals

## Guiding question
What are the principles and concepts that underpin how networks operate?

---

### A2.1.1 · Describe the purpose and characteristics of networks
- Network types: local area network (LAN), wide area network (WAN), personal area network (PAN), virtual private network (VPN).

### A2.1.2 · Describe the purpose, benefits, and limitations of modern digital infrastructures
- Infrastructure examples: internet, cloud computing, distributed systems, edge computing, mobile networks.
- Apply examples such as WWW, blockchain, smart traffic systems, schools.

### A2.1.3 · Describe the function of network devices
- Gateways, hardware firewalls, modems, NICs, routers, switches, wireless access points.
- Map devices to TCP/IP layers.

### A2.1.4 · Describe transport and application layer protocols
- Protocols: TCP, UDP, HTTP, HTTPS, DHCP.

### A2.1.5 · Describe the function of the TCP/IP model (HL only)
- Layers: application, transport, internet, network interface.
- Explain interactions that ensure reliable transmission.
