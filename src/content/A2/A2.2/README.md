# A2.2 · Network architecture

### A2.2.1 · Describe the functions and practical applications of network topologies
- Topologies: star, mesh, hybrid.
- Evaluate reliability, speed, scalability, collisions, cost with example scenarios.

### A2.2.2 · Describe the function of servers (HL only)
- Server types: DNS, DHCP, file, mail, proxy, web.
- Consider function, scalability, reliability, security.

### A2.2.3 · Compare and contrast networking models
- Client–server vs peer-to-peer.
- Discuss benefits, drawbacks, and applications (web, email, banking, file sharing, VoIP, blockchain).

### A2.2.4 · Explain network segmentation
- Purposes: performance, security, congestion reduction, resource management.
- Techniques: segmentation, subnetting, VLANs.
