# B2.1 · Programming fundamentals

### B2.1.1 · Construct and trace programs with variables
- Use global/local variables; data types: Boolean, char, decimal, integer, string.

### B2.1.2 · Manipulate substrings
- Implement routines to extract, alter, concatenate, replace substrings.

### B2.1.3 · Describe exception handling
- Identify failure points; apply try/catch (Java) or try/except (Python) with finally blocks.

### B2.1.4 · Construct debugging techniques
- Use trace tables, breakpoints, logging/print statements, step-by-step execution.
