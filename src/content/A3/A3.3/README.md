# A3.3 · Database programming

### A3.3.1 · Outline SQL language types
- Distinguish DDL vs DML.

### A3.3.2 · Construct queries between two tables
- Include joins, relational operators, filtering, pattern matching, ordering.
- Commands: SELECT, DISTINCT, FROM, WHERE, BETWEEN, ORDER BY, GROUP BY, HAVING, ASC, DESC, JOIN, LIKE (% wildcard), AND, OR, NOT.

### A3.3.3 · Explain SQL data updates
- INSERT INTO, UPDATE SET, DELETE; effects on indexes and maintenance.

### A3.3.4 · Construct aggregate calculations (HL only)
- AVERAGE, COUNT, MAX, MIN, SUM on grouped data.

### A3.3.5 · Describe database views (HL only)
- Virtual vs materialised views; hiding complexity, performance, security.

### A3.3.6 · Describe transactions and ACID (HL only)
- Atomicity, consistency, isolation, durability; TCL commands BEGIN, COMMIT, ROLLBACK.
