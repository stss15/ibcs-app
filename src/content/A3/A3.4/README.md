# A3.4 · Alternative databases and data warehouses (HL only)

### A3.4.1 · Outline alternative database models
- NoSQL, cloud, spatial, in-memory databases with example use cases (e-commerce, GIS, managed services, real-time analytics, social media, SaaS).

### A3.4.2 · Explain objectives of data warehouses
- Characteristics: append-only, subject-oriented, integrated, time-variant, non-volatile, optimised for queries.

### A3.4.3 · Explain OLAP and data mining for business intelligence
- Techniques: classification, clustering, regression, association rules, sequential patterns, anomaly detection.
- Link outputs to decision-making and to A4 machine learning content.

### A3.4.4 · Describe features of distributed databases
- Address: consistency, ACID transactions, concurrency control, partitioning, replication, distribution transparencies, fault tolerance, global query processing, scalability, security.
