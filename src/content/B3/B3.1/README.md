# B3.1 · Fundamentals of object-oriented programming (single class)

### B3.1.1 · Evaluate OOP fundamentals
- Model real-world entities with classes, objects, inheritance, encapsulation, polymorphism; assess pros/cons.

### B3.1.2 · Design classes, methods, behaviour
- Use UML class diagrams to plan attributes, methods, relationships.

### B3.1.3 · Distinguish static vs non-static members
- Describe usage and scope for class-level vs instance-level state/behaviour.

### B3.1.4 · Define classes and instantiate objects
- Explain constructor roles in initialising state.

### B3.1.5 · Explain encapsulation and information hiding
- Apply access modifiers (private/public) to protect object state and enforce interfaces.
