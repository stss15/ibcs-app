## Learning intentions
- Categorise storage devices as internal (SSD, HDD, eMMC) or external (SSD, HDD, optical, flash, memory cards, NAS).
- Match storage types to appropriate usage scenarios considering capacity, portability, durability.
- Compare performance characteristics and interfaces (SATA, NVMe, USB, Thunderbolt, network protocols).

## Teaching notes
- Bring physical devices or images to support tactile understanding.
- Include discussion on redundancy/backups and role of NAS in school environments.
