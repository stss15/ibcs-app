## Learning intentions
- Analyse multitasking challenges: scheduling, resource contention, synchronisation, deadlock.
- Explain OS strategies for allocating CPU, memory, and I/O resources among competing processes.
- Discuss mechanisms such as mutexes, semaphores, priority inversion handling.

## Teaching notes
- Use case studies (e.g., scheduling audio/video threads alongside background tasks) to ground discussion.
- Introduce classic problems (Dining Philosophers, Producer/Consumer) that illustrate contention.
